# tinydash

The `tinydash` A small JavaScript utility library.

## Basic Use

npm

```shell
npm i tinydash
```

pnpm

```shell
pnpm add tinydash
```

## Example

```javascript
import { isNaN } from 'tinydash'

console.log(isNaN(1)); // false
console.log(isNaN(NaN)); // true
```
