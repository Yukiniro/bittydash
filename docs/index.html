The bittydash A small JavaScript utility library.

Basic Use

npm

npm i bittydash

pnpm

pnpm add bittydash

Example

 
import { isNaN } from 'bittydash'

console.log(isNaN(1)); // false
console.log(isNaN(NaN)); // true
